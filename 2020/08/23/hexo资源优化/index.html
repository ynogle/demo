hexo资源优化

1.hexo-neat
npm install hexo-neat –save

项目配置文件下添加

hexo-neat

博文压缩

neat_enable: true

压缩html

neat_html:
enable: true
exclude:

压缩css

neat_css:
enable: true
exclude:
- ‘**/*.min.css’

压缩js

neat_js:
enable: true
mangle: true
output:
compress:
exclude:
- ‘/*.min.js’
- ‘/jquery.fancybox.pack.js’
- ‘**/index.js’
